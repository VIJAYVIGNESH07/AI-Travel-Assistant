# Animation Specifications

**Splash**
- Logo scale spring from 0.8 to 1.0
- Duration approximately 2.2 seconds

**Buttons**
- Press scale to 0.98

**Chat**
- Message bubbles fade in

**Navigation**
- Stack screens slide from right
- Modal screens fade

Reanimated is set up in `babel.config.js` and ready for richer motion.
